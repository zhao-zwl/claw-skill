# HEARTBEAT.md

```markdown
# Keep this file empty (or with only comments) to skip heartbeat API calls.

# Add tasks below when you want the agent to check something periodically.
```

## 模板说明

保持空或只有注释 = 不做任何周期性检查（节省 API 调用）。

如需定期执行任务，在上方添加即可，例如：

```markdown
## 每日检查
- [ ] 检查邮件（早上 9 点）
- [ ] 更新任务看板
```

每个任务执行后记得**立即汇报主控**，不要等。
